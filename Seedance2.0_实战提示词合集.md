## 一、PVE 战斗类

### 1. 水墨国风战场

**风格**：中国水墨 | **色调**：黑白灰 + 红绿点缀 | **类型**：PVE 围攻

```
中国水墨风格战场，
宣纸质感背景，
山水泼墨远景，
留白空间明显，
墨色云雾在空中翻涌，
地面呈现水墨晕染纹理，
整体黑白灰为主色调，
局部点缀红色与绿色。

水墨晕染扩散效果，
墨点飞溅形成动态笔触，
书法笔锋拖尾，
留白构图，
国风鼓点节奏。

【主角】作为主角，
头顶葫芦叶微微晃动，
表情坚定。

大量【蛤蟆敌人】从墨色烟雾中跳出，
黄黑花纹在水墨中显得突兀，
动作夸张跳跃，
落地时溅起墨汁效果，
数量密集成群围攻。

镜头快速侧移跟拍，
【主角】冲刺滑步，
脚踩地面溅起黑色墨痕，
翻滚闪避蛤蟆扑击。

近战拳击命中，
【蛤蟆敌人】被击飞，
身体在空中化作泼墨残影，
落地炸开墨花。

镜头环绕【主角】旋转，
【蛤蟆敌人】从四面八方跃起，
【主角】腾空旋转踢击，
墨色轨迹在空中形成弧线。

节奏突然压缩，
镜头拉近【主角】眼神特写，
背景墨云翻滚。

大招爆发——
【主角】双拳落地，
地面墨色迅速扩散成巨大圆形水墨阵法，
墨汁化为冲击波向外席卷，
【蛤蟆敌人】被卷入墨浪之中，
身体碎散成黑色墨点。

强烈白色留白闪现，
随后墨迹重新汇聚。

镜头俯拍，
战场中央只剩【主角】站立，
身后墨山轮廓浮现，
绿色与红色在画面中格外醒目，
画面定格在国风英雄姿态。
```

### 2. 熔岩火山战场

**风格**：熔岩史诗 | **色调**：暗红橙 + 紫色 | **类型**：PVE 围攻

```
熔岩火山战场环境，
暗红色天空，
远处火山喷发，
炽热熔岩在地面裂缝中流动，
岩石表面发出橙红色光芒，
火山灰缓慢飘落，
空气扭曲发热。

整体光源来自熔岩，
橙红色光影照亮角色轮廓，
强烈背光制造剪影效果，
地面不断喷出火花。

【主角】，
双马尾在热浪中剧烈摆动，
护目镜反射火光，
手持未来感能量手枪。

大量Q版骷髅小兵从岩石后爬出，
骨架比例夸张，
空洞眼窝发出微弱红光，
动作灵活快速，
数量密集成群包围。

镜头高速侧移跟拍，
【主角】滑步冲刺，
跳跃踩踏骷髅头顶，
腾空翻转，
双枪同时开火。

紫色能量弹道划破火山灰，
击中骷髅小兵，
骨架被震散，
骨头碎片飞溅。

镜头低角度仰拍，
【主角】在空中旋转射击，
熔岩背景在身后翻滚，
爆炸冲击波震散骷髅群。

骷髅从四周再次涌上，
【主角】落地滑步，
近战枪托击碎骷髅头，
旋身踢飞成排敌人。

镜头快速环绕360度旋转，
骷髅小兵再次从四周涌来，
数量远超之前，
地面不断裂开，
熔岩翻滚喷发，
火山灰遮蔽天空，
画面压迫到极限。

镜头突然快速推进【主角】眼神特写，
护目镜反射火焰，
瞳孔微微收缩，
背景声音骤然压低。

时间短暂停滞。

下一瞬间，
【主角】双枪高举，
紫色与橙红色能量在空中汇聚，
形成巨大的旋转能量漩涡。

镜头绕人物高速环绕，
双马尾与裙摆在能量风暴中狂舞，
熔岩被吸起悬浮在空中，
碎石围绕她旋转。

【主角】腾空跃起，
空中翻转，
双枪向下交叉射击。

巨大的能量光柱轰然落地，
形成环形冲击波，
熔岩被瞬间推开，
骷髅小兵在冲击中成片粉碎。

强烈白光闪烁，
火山背景被能量撕裂，
冲击波席卷整个战场。

光芒散去，
镜头从高空俯拍缓慢落下，
地面布满碎骨与裂纹，
熔岩流动在远处缓缓燃烧。

【主角】单膝落地，
双枪缓缓旋转收回，
背对火山，
火光勾勒出炽热剪影。

镜头缓慢推近，
【主角】轻轻抬头，
嘴角微扬，
画面定格在英雄式帅气姿态。
```

### 3. 末日城市街区

**风格**：末日赛博 | **色调**：冷灰蓝 + 紫粉霓虹 | **类型**：PVE 围攻

```
末日城市街区场景，
冷色调灯光与紫蓝色霓虹反射，
废弃车辆与残破建筑，
空气中弥漫薄雾与尘埃，
整体氛围紧张压迫。

【主角】双马尾卷发随动作剧烈摆动，
佩戴护目镜，
手持未来感能量手枪。

镜头高速侧移跟拍，
【主角】突然滑步冲刺，
贴地翻滚躲开丧尸扑击，
动作干净利落。

镜头快速切到低角度仰拍，
她腾空跃起，
空中旋转，
双枪同时开火，
紫色能量弹道划出弧线轨迹，
Q版丧尸被击飞。

镜头环绕360度旋转，
子弹轨迹在空中形成光圈，
爆炸产生卡通冲击波，
丧尸成片倒下。

【主角】落地瞬间，
滑步转身，
近距离枪托击打，
接一个旋转踢腿，
动作连贯流畅。

镜头突然推近面部特写，
护目镜反射火光，
眼神锐利。

下一秒，
【主角】向前冲刺穿过丧尸群，
镜头跟随高速推进，
残影拖尾明显，
背景速度线填满画面。

最后一记空中后翻落地，
爆发范围能量冲击，
丧尸全部震飞。

镜头从高处俯拍，
【主角】站在战场中央，
烟雾翻涌，
双枪缓缓旋转入手，
画面定格在酷炫战斗姿态。
```

```
日本热血动画风格，
高对比色彩，鲜明轮廓线条，
流畅的二维动画质感，
动态速度线与能量特效明显，
画面节奏快速。

夜晚的忍者森林场景，
月光穿过树叶形成斑驳光影，
地面落叶被风吹起，
空气中弥漫紧张气氛。

【角色A】和【角色B】对峙，
身穿忍者服装，
护额与披风随风摆动，
镜头快速切换到眼神特写，
瞳孔收缩，杀气凝聚。

瞬间消失，
高速移动残影出现，
地面被踏裂，
树木被气流震动。

查克拉能量在手中聚集，
蓝色或红色能量光爆发，
电流或火焰特效缠绕手臂，
镜头环绕角色旋转。

高速近身交锋，
拳脚碰撞产生冲击波，
速度线贯穿背景，
画面出现短暂慢动作展示关键一击。

【角色A】腾空翻转，
结印动作快速完成，
释放强力忍术，
能量冲击席卷森林。

烟雾散去，
镜头拉远，
【角色A】站立在月光下，
披风缓缓落下，
画面定格在热血英雄姿态。
```

日本热血动画风格，
高对比色彩，强烈光效，

荒凉岩石平原场景，
天空阴云翻滚，
远处山脉被风压笼罩，
空气中充满能量波动。

【角色A】、【角色B】在高空对峙，
金色或绿色能量气焰包裹全身，
气流向外扩散，
地面碎石被无形力量震起。

突然瞬移，
高速残影在空中交错，

空气被压缩形成圆形气浪，
地面被震裂。

低角度仰拍增强力量感，
连续重拳与踢击，

烟雾散去，

## 三、追逐/生存类

### 6. 巨蟒密林追逐（时间轴分镜）

**风格**：日式热血动漫 | **色调**：深绿蓝灰冷调 | **类型**：追逐/生存 | **结构**：时间戳分镜

```
日式热血动漫风格，
高对比度光影，
鲜明轮廓线，
夸张透视，
大量速度线，
动态模糊明显，
高张力作画，
集中线爆发效果。

幽暗密林环境，
深绿色与蓝灰色冷色调，
高大古树遮蔽天空，
光线从树缝穿透，
地面落叶与薄雾弥漫。

0.0s–1.0s，
极近特写，
巨蟒竖瞳特写，
瞳孔中倒映出渺小的【主角】跑步背影，
镜头急速推入瞳孔，
瞬间黑场过渡。

1.0s–2.2s，
大远景鸟瞰，
森林如绿色海洋，
一道巨大黑色轨迹在林中高速蜿蜒，
树木被成排撞飞，
高空跟拍巨蟒移动。

2.2s–3.6s，
中景侧面跟拍，
【主角】高速奔跑，
背景转为绿色速度线，
汗水飞溅，
呼吸急促。

3.6s–4.8s，
过肩镜头，
【主角】表情惊恐害怕，
后方巨蟒张开血盆大口，
紫色腥气烟雾弥漫，
镜头带手持晃动感。

4.8s–6.0s，
主观视角（巨蟒视角），
高速俯冲逼近【主角】，
画面边缘强烈动态模糊，
猎食锁定感明显。

6.0s–7.2s，
特写转全景，
【主角】猛地滑铲，
慢动作一瞬，
巨蟒头部擦着【主角】撞向地面，
泥土碎石炸裂，
随后恢复高速。

7.2s–8.8s，
仰拍特写，
烟尘弥漫，
【主角】半跪在地，
巨大蛇身横亘画面上方。

8.8s–10.5s，
中远景，
巨蟒缓缓抬起上半身，
体量压过整片树林，
光线被完全遮蔽，
镜头缓慢推向蛇头，
空气凝滞。

10.5s–13.0s，
特写到极近特写，
巨蟒突然张开巨大口器，
集中线由四周向中心爆发，
镜头急速推进，
巨口占满画面，
镜头剧烈震动，
瞬间黑场。
```

## 四、提线木偶创意类

### 7.「偷吃」— 木偶嘴是画上去的

**风格**：提线木偶定格动画 | **色调**：暖黄柔光 + 木质纹理 | **类型**：趣味短片

> **素材准备说明**：
> - 图片素材：葫芦娃木偶多角度参考图（正面/侧面/背面）、紫色怪兽木偶参考图
> - 视频素材：提线木偶运动方式参考视频（展示关节晃动、线绳牵拉的典型运动模式）
> - 使用 Seedance 2.0 多模态输入：图片作为角色参考，视频作为运动风格参考

```
提线木偶定格动画风格，
木质雕刻纹理清晰可见，
关节处有明显的球形铰链连接，
角色头顶悬挂细线连接至画面上方的木质操控十字架，
整体暖黄色柔和灯光，
背景为木质小剧场舞台，
手工感布艺幕布。

镜头从舞台上方缓缓下摇，
画面顶部可见一双巨大的真人手正握着木质操控架，
手指微微拨动，
线绳轻轻晃动。

@葫芦娃木偶 被线绳牵引着从舞台侧方摇摇晃晃走出，
木质关节咔咔作响，
步伐带有典型的提线木偶僵硬弹跳感，
头部因为线绳拉扯微微左右摆动。

舞台桌上摆着一个巨大的糖葫芦道具，
@葫芦娃木偶 看到后停下，
脑袋猛地转向糖葫芦（头部旋转角度略微夸张），
身体僵住一秒。

@葫芦娃木偶 左看看右看看（整个身体跟着头一起转动，因为是木偶），
确认没人，
木质手臂僵硬地抬起，
一把抓住糖葫芦举到脸前。

关键一刻——
@葫芦娃木偶 把糖葫芦怼到脸上，
但嘴巴是画上去的根本张不开，
糖葫芦反复怼在画出来的嘴唇上，
发出木头撞击的"哒哒哒"声，
头部被撞得前后摇晃，
线绳跟着剧烈抖动。

画面上方的大手突然抖了一下（操控者在笑），
@葫芦娃木偶 整个身体失控地跟着晃荡了几下。

@葫芦娃木偶 不死心，
换个角度继续怼，
侧面怼、仰头怼，
动作越来越急促，
每次都是木头碰木头的硬邦邦声音，
糖葫芦上的糖浆蹭了木偶一脸。

最后 @葫芦娃木偶 整个身体僵住，
低头看着糖葫芦，
虽然表情是固定画上去的，
但在灯光微妙变化下竟显出一丝哀伤。

这时 @紫色怪兽木偶 从另一侧摇晃着上场，
张着巨大的（可以活动的）嘴巴，
一口叼走了糖葫芦，
嘎吱嘎吱地嚼，
嘴巴开合的木质铰链咔嗒作响。

@葫芦娃木偶 愣住，
然后缓缓转头看向镜头（打破第四面墙），
线绳无力地垂下几分。

画面上方的大手轻轻晃了晃线绳，
像在安慰。

镜头缓慢拉远，
暖黄灯光微微变暗，
整个小剧场舞台尽收眼底，
幕布缓缓合拢。

木质碰撞声与铰链咔嗒声贯穿全程，
背景配乐为轻快的手风琴小调，
偷吃失败瞬间音乐突然停顿，
最后一个音符拖长消散。
```
